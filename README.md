# Kritika_Store_Annual_Report_Excel_Dashboard
An Excel project to analyze Annual Report for Kritika Store

This project is for beginners and will teach you how to analyze the Any Store database. You can examine the dataset with EXCEL Using Pivot Chart, Pivot Table, Slicer and Create an interactive Dashboard which help the store to understand its business growth by answering simple questions.

Project helps you with Data Cleaning, Data Processing, Data Analysis, Data Visualization, Preparing Dashboard & Report and Share Insight.
